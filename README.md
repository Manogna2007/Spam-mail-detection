# 📧 Spam Mail Detection

A Machine Learning project that classifies emails as **Spam** or **Ham (Not Spam)** using **Logistic Regression** and **TF-IDF Vectorization**.

## 🚀 Features
- Email classification into Spam or Ham
- Text feature extraction using TF-IDF
- Logistic Regression model
- Model evaluation using Accuracy Score

## 🛠️ Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- Jupyter Notebook

## 📂 Project Files
- `spam_mail_detection.ipynb` – Complete model implementation
- `mail_data.csv` – Dataset

## ▶️ How to Run

```bash
git clone https://github.com/Manogna2007/Spam-mail-detection.git
cd Spam-mail-detection
pip install numpy pandas scikit-learn
jupyter notebook
```

Run all the notebook cells to train the model and make predictions.
